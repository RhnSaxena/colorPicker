# colorPicker

## Tech Stack

- Python
- Flask

## Getting Started

Steps:

- Install the requirements
  ```
  pip install -r requirements.txt
  ```
- Please configure the port before moving to the next step.
  In ./config.py

  ```
  host = "127.0.0.1"
  port=5007
  ```

- Start the app
  ```
  python ./main.py
  ```
